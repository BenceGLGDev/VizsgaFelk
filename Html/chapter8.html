8. Reszponzív Design

Mi az a Reszponzív Webdesign?

A reszponzív design célja, hogy a weboldalak minden eszközön jól nézzenek ki és megfelelően működjenek.

Viewport Beállítása

A reszponzív tervezés egyik alapja a meta viewport beállítás.


<meta name="viewport" content="width=device-width, initial-scale=1.0">

CSS Media Query-k

Nem szerepel tudomásom szerint, de baromi hasznos.

A média lekérdezések segítségével a stílusokat különböző képernyőméretekhez igazíthatjuk.


@media (max-width: 600px) {
    body {
        background-color: lightgray;
    }
}

Reszponzív Képek

A képek méretének automatikus beállítása fontos a reszponzív designhoz.


img {
    max-width: 100%;
    height: auto;
}

Flexbox és Grid

A modern CSS elrendezési technikák, mint a Flexbox és a Grid, segítenek reszponzív weboldalak létrehozásában.


.container {
    display: flex;
    flex-wrap: wrap;
    justify-content: space-between;
}

Gyakorló Feladatok:

  1. Állítsd be a meta viewport elemet a HTML fájlodban.
  2. Készíts egy egyszerű media query-t, amely változtatja a háttérszínt kisebb képernyőkön.
  3. Használj flexboxot egy három oszlopból álló reszponzív elrendezés létrehozására.
