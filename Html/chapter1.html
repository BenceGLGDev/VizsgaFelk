1. Bevezetés a HTML-be

Mi az a HTML?

A HTML (HyperText Markup Language) a weblapok alapvető nyelve, amely meghatározza a weboldalak szerkezetét és tartalmát.

  • A HTML nem egy programozási nyelv, hanem egy jelölőnyelv.
  • Lehetővé teszi szövegek, képek, linkek és egyéb elemek megjelenítését a weboldalon.

HTML Alapstruktúrája

Minden HTML dokumentum egy adott szerkezetet követ. Íme egy egyszerű HTML oldal:


<!DOCTYPE html>
<html>
<head>
    <title>Első Weboldalam</title>
</head>
<body>
    <h1>Üdvözöllek a weboldalamon!</h1>
    <p>Ez az első HTML oldalam.</p>
</body>
</html>

HTML Elemei

A HTML dokumentum különböző elemekből áll, amelyeket címkék (<tag>) határoznak meg.

  • <html> - A dokumentum kezdete és vége.
  • <head> - A dokumentum fejléce (pl. cím, meta adatok).
  • <body> - A dokumentum fő tartalma.
  • <h1> - Főcím.
  • <p> - Bekezdés.

Gyakorló Feladatok:

  1. Hozz létre egy HTML fájlt, amely tartalmaz egy főcímet és egy bekezdést.
  2. Módosítsd a címet és a szöveget saját tartalommal!
  3. Próbálj meg további HTML elemeket hozzáadni!
