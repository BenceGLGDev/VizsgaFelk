5. Flexbox

Mi az a Flexbox?

A Flexbox (Flexible Box Layout) egy erőteljes eszköz, amely lehetővé teszi a HTML elemek hatékony és dinamikus elrendezését egy dimenzióban (sor vagy oszlop).

Alapvető Flexbox tulajdonságok

  • display: flex; – Flexbox bekapcsolása egy szülőelemre.
  • flex-direction – A tengely irányának beállítása (row, column stb.).
  • justify-content – Az elemek igazítása a főtengely mentén.
  • align-items – Az elemek igazítása a keresztirányú tengely mentén.
  • flex-wrap – Meghatározza, hogy az elemek törjenek-e több sorba.

Példa Flexbox használatára

.container {
    display: flex;
    flex-direction: row;
    justify-content: center;
    align-items: center;
    height: 300px;
    background-color: lightgray;
}

Gyakorló Feladatok:

  1. Hozz létre egy rugalmas konténert display: flex; használatával!
  2. Igazítsd az elemeket középre a főtengely mentén!
  3. Állítsd be az elemeket, hogy függőlegesen középre kerüljenek!
