6. Grid

Mi az a CSS Grid?

A CSS Grid egy kétdimenziós elrendezési rendszer, amely lehetővé teszi a tartalom rugalmas és hatékony elhelyezését sorokban és oszlopokban.

Alapvető Grid tulajdonságok

  • display: grid; – Bekapcsolja a grid elrendezést.
  • grid-template-columns – Oszlopok meghatározása.
  • grid-template-rows – Sorok meghatározása.
  • gap – Az elemek közötti térköz beállítása.
  • grid-column és grid-row – Egy elem adott oszlophoz vagy sorhoz rendelése.

Példa Grid használatára

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: auto;
    gap: 10px;
    background-color: lightgray;
}

Gyakorló Feladatok:

  1. Hozz létre egy 3 oszlopból álló grid konténert!
  2. Állíts be egy 10px-es gap-et az elemek között!
  3. Helyezz el egy elemet, amely 2 oszlop szélességű!
