3. Box Model

Mi az a Box Model?

A CSS Box Model minden HTML elemet egy dobozként kezel. Négy fő részből áll:

  • Content (Tartalom) – Az elem tényleges tartalma (szöveg, kép stb.).
  • Padding – A tartalom és a keret közötti tér.
  • Border – A dobozt körülvevő keret.
  • Margin – Az elem körüli külső tér, amely elválasztja más elemektől.

Box Model felépítése

div {
    width: 200px;
    height: 100px;
    padding: 20px;
    border: 5px solid black;
    margin: 10px;
}

Példa Box Modelre

Az alábbi példában látható, hogyan befolyásolják a különböző tulajdonságok a doboz méretét:

.box {
    width: 300px;
    padding: 20px;
    border: 10px solid blue;
    margin: 30px;
    background-color: lightgray;
}

Gyakorló Feladatok:

  1. Adj egy 15px-es padding-ot egy div elemhez!
  2. Állíts be egy 5px-es piros színű border-t!
  3. Növeld a margin-t 25px-re egy bekezdésen!
