7. Reszponzív Design

Mi az a Reszponzív Design?

A reszponzív webdesign lehetővé teszi, hogy a weboldalak különböző képernyőméreteken és eszközökön jól nézzenek ki és megfelelően működjenek.

Alapvető technikák

  • Rugalmas rácsrendszerek: A CSS Grid és Flexbox használata az elrendezés rugalmas kezelésére.
  • Reszponzív képek: max-width: 100% beállítás használata a képek megfelelő méretezéséhez.
  • Média lekérdezések: CSS @media szabályok alkalmazása különböző képernyőméretekhez.

Példa Média lekérdezésre

@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}

Gyakorló Feladatok:

  1. Készíts egy @media szabályt, amely 600px alatti kijelzőknél módosítja az elrendezést!
  2. Használj rugalmas képeket a max-width: 100% tulajdonsággal!
  3. Próbálj ki egy CSS Grid vagy Flexbox elrendezést, amely reszponzívan alkalmazkodik!
