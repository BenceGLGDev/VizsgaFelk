4. Pozicionálás

Mi az a CSS Pozicionálás?

A CSS-ben az elemek elhelyezkedését a position tulajdonsággal szabályozhatjuk.

  • static – Alapértelmezett érték, az elem a normál dokumentumfolyamban marad.
  • relative – Az elem a saját eredeti helyzetéhez képest mozgatható.
  • absolute – Az elem az első relative vagy absolute ős elemhez képest pozicionálódik.
  • fixed – Az elem a nézetablakhoz rögzítve marad görgetéskor is.
  • sticky – Az elem a görgetés során egy meghatározott pozícióhoz tapad.

Példa Pozicionálásra

.box {
    position: relative;
    top: 20px;
    left: 50px;
    background-color: lightblue;
    padding: 10px;
}

Gyakorló Feladatok:

  1. Helyezd el egy elem absolute pozícióval a jobb felső sarokba!
  2. Adj egy elemnek fixed pozíciót, hogy mindig látható maradjon!
  3. Készíts egy sticky navigációs sávot!
