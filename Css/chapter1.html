1. Bevezetés a CSS-be

Mi is az a CSS?

A CSS (Cascading Style Sheets) a weboldalak megjelenésének szabályozására szolgál. Segítségével beállíthatjuk a színeket, betűtípusokat, elrendezést és animációkat.

Miért fontos a CSS?

  • Elválasztja a tartalmat és a megjelenést.
  • Egyszerűsíti a weboldalak karbantartását.
  • Reszponzív designt tesz lehetővé.

CSS szintaxis

selector {
    property: value;
}

CSS használatának módjai

Módszer Leírás
Inline CSS Elemekhez egyenként alkalmazható a style attribútummal.
Belső CSS A <style> blokkban az HTML fájlon belül.
Külső CSS Külön fájlban (.css) tárolt stíluslap.

Példa egy egyszerű CSS-re

body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
}

Gyakorló Feladatok:

  1. Állítsd be a háttérszínt sárgára az body elemben!
  2. Változtasd meg a szöveg színét pirosra!
  3. Adj hozzá egy külső CSS fájlt a weboldalhoz!
