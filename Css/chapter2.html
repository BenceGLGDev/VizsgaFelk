2. Színek és Betűtípusok

CSS színek

A CSS-ben többféle módon adhatunk meg színeket:

  • Nevek (pl. red, blue)
  • Hexadecimális kódok (pl. #ff0000)
  • RGB értékek (pl. rgb(255, 0, 0))
  • HSL értékek (pl. hsl(0, 100%, 50%))

Példa színek használatára

body {
    background-color: lightgray;
    color: navy;
}

CSS betűtípusok

A betűtípusokat a font-family tulajdonsággal állíthatjuk be.

Példa betűtípusok használatára

p {
    font-family: Arial, sans-serif;
    font-size: 16px;
    font-weight: bold;
    font-style: italic;
}

Gyakorló Feladatok:

  1. Állítsd be a háttérszínt zöldre!
  2. Változtasd meg az összes szöveg színét kékre!
  3. Használj egyedi betűtípust az oldal főcíméhez!
