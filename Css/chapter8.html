8. Animációk és Átmenetek

Miért fontosak az animációk?

A CSS animációk és átmenetek vizuálisan vonzóbbá teszik a weboldalakat, és segítenek a felhasználói élmény javításában.

CSS Átmenetek

Az átmenetek lehetővé teszik az elemek tulajdonságainak zökkenőmentes változását.

button {
    background-color: blue;
    transition: background-color 0.3s ease;
}

button:hover {
    background-color: red;
}

CSS Animációk

Az animációk segítségével komplexebb mozgásokat és változásokat valósíthatunk meg.

@keyframes fadeIn {
    from {
        opacity: 0;
    }
    to {
        opacity: 1;
    }
}

.element {
    animation: fadeIn 2s ease-in;
}

Gyakorló Feladatok:

  1. Hozz létre egy gombot, amely színátmenetet kap hover eseményre!
  2. Animálj egy elemet, hogy fokozatosan jelenjen meg a képernyőn!
  3. Készíts egy saját @keyframes animációt, amely mozgat egy elemet!
