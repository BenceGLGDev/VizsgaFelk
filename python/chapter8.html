8. Függvények és Eljárások

Függvények és eljárások fogalma

A Pythonban bármikor készíthetünk saját függvényeket, amelyek megkönnyítik a programozási folyamatokat. Függvényt a def kulcsszóval készítünk.

  • Függvény: Ha van visszatérési értéke (return).
  • Eljárás: Ha nincs visszatérési értéke (csak végrehajt valamit).
def koszont():
    print("Üdv a Python kurzuson!")

Paraméterek átadása

A függvények paramétereket fogadhatnak, amelyeket változóként használhatunk a függvényen belül. A paramétereket vesszővel választjuk el.

def osszeadas(a, b):
    return a + b

print(osszeadas(3, 5))  # 8

Default értékek

A paramétereknek alapértelmezett értékeket is megadhatunk, amelyek akkor érvényesek, ha a függvényt ezen paraméter nélkül hívjuk meg:

def koszont(nev="Vendég"):
    print(f"Üdv, {nev}!")

koszont()  # Üdv, Vendég!
koszont("Anna")  # Üdv, Anna!

Ismeretlen paraméterszám

Ha nem tudjuk előre, hogy hány paramétert fogad a függvény, a következő megoldásokat használhatjuk:

*args

A *args (jelen esetben "szamok") használatával tetszőleges számú paramétert adhatunk meg listaként:

def osszeg(*szamok):
    return sum(szamok)

print(osszeg(1, 2, 3, 4))  # 10

**kwargs

A **kwargs (jelen esetben "adatok") segítségével tetszőleges számú kulcs-érték párt adhatunk meg:

def info(**adatok):
    for kulcs, ertek in adatok.items():
        print(f"{kulcs}: {ertek}")

info(nev="Anna", kor=25, varos="Budapest")

Visszatérési érték

A függvény visszatérési értéke a return kulcsszóval adható meg:

def negyzet(x):
    return x ** 2

print(negyzet(4))  # 16

Rekurzió

Rekurzió esetén egy függvény önmagát hívja meg egy adott feltételig. Példa:

def faktorialis(n):
    if n == 1:
        return 1
    else:
        return n * faktorialis(n - 1)

print(faktorialis(4))  # 24

Fontos: Óvatosan kell használni a rekurziót, mivel könnyen végtelen ciklust vagy túlzott memóriahasználatot eredményezhet!
