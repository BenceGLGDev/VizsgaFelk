5. Random Modul

Modulok Importálása

A modul egy olyan fájl, amely funkciókat és értékeket tartalmaz, amelyekre hivatkozni lehet a programból.

Az importálás szintaxisa:

import modulnev

Beépített Modulok

A Python több beépített modult is biztosít, például:

  • random: véletlen műveletekre (véletlen szám, lista keverése stb.).
  • math: matematikai műveletekre (gyökvonás, hatványozás stb.).

A Random Modul

A random modul véletlen műveletekre használható. A használathoz importálni kell a modult:

import random

Véletlen Szám Generálás

Egész számok:

Egész szám generálása a randint függvénnyel:

import random
szam = random.randint(1, 10)
print(szam)  # 1 és 10 közötti véletlen szám

Alternatíva: randrange:

szam = random.randrange(1, 10)
print(szam)  # 1 és 9 közötti véletlen szám

Lebegőpontos számok:

Lebegőpontos szám generálása 0.0 és 1.0 között:

szam = random.random()
print(szam)

Két szám közötti lebegőpontos szám generálása:

szam = random.uniform(1.5, 5.5)
print(szam)  # 1.5 és 5.5 közötti véletlen szám

Lista Műveletek

Véletlen elem kiválasztása:

lista = ["alma", "körte", "barack"]
elem = random.choice(lista)
print(elem)  # Egy véletlen elem a listából

Véletlen több elem kiválasztása:

elemek = random.sample(lista, 2)
print(elemek)  # Két véletlen elem a listából

Lista összekeverése:

A lista elemeit véletlenszerű sorrendbe állíthatjuk:

random.shuffle(lista)
print(lista)  # A lista véletlenszerűen összekeverve
