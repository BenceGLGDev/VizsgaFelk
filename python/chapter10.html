10. Objektumorientált Programozás (OOP)

Python Objektumok és Osztályok

A Pythonban szinte minden objektum, amely beállításokat és metódusokat tartalmaz. Az objektumokat úgy kell tekinteni, mint egy tervrajzot, amely alapján példányosítással objektumokat hozhatunk létre.

Osztály Létrehozása

Az osztály létrehozásához a class kulcsszót használjuk:

class Ember:
    nev = ""
    kor = 0

    def koszont(self):
        print(f"Szia, {self.nev}!")

Objektum Létrehozása

Egy osztály alapján objektumokat hozhatunk létre:

ember1 = Ember()
ember1.nev = "Anna"
ember1.kor = 25
ember1.koszont()

Az __init__() Metódus

Az __init__() metódus egy speciális metódus, amely automatikusan meghívódik objektum létrehozásakor. Ezzel lehet beállítani az alapértelmezett értékeket:

class Ember:
    def __init__(self, nev, kor):
        self.nev = nev
        self.kor = kor

    def koszont(self):
        print(f"Szia, {self.nev}!")

ember1 = Ember("Anna", 25)
ember1.koszont()

A self Paraméter

A self paraméter az aktuális példányra való hivatkozás, amelyen keresztül elérhetők az osztály változói. Ez a paraméter tetszőleges névvel helyettesíthető, de a self a szokásos konvenció.

Érték Módosítása

Egy objektum értékeit közvetlenül vagy metódusokon keresztül módosíthatjuk:

ember1.kor = 26
print(ember1.kor)

Az del Kulcsszó

Az del kulcsszóval törölhetjük az osztály egy attribútumát vagy magát az objektumot:

del ember1.kor
# Objektum törlése
# del ember1

A pass Kulcsszó

Ha egy osztály üres, a pass kulcsszó használatával elkerülhetjük a hibát:

class UresOsztaly:
    pass
