7. Tuple és Szótárak

Tuple adattípus

A tuple egy olyan adattípus, amely több értéket képes egyszerre tárolni. Jellemzői:

  • Nem módosítható (immutable).
  • A tuple elemeinek sorrendje rögzített.
  • Indexelés segítségével érhetők el az elemek.
my_tuple = ("alma", "körte", "barack")
print(my_tuple[0])  # alma

Tuple metódusai

  • index(): Megkeresi egy érték első előfordulási indexét.
  • count(): Megszámolja, hogy hányszor fordul elő egy adott érték.
my_tuple = ("alma", "körte", "alma")
print(my_tuple.count("alma"))  # 2
print(my_tuple.index("körte"))  # 1

Dictionary (Szótár)

A dictionary kulcs-érték párokat tartalmazó adatszerkezet. Jellemzői:

  • Indexek helyett kulcsokat használ.
  • A kulcsok egyediek (nincs duplikáció).
  • A kulcsok és az értékek bármilyen adattípusúak lehetnek.
my_dict = {
    "nev": "Anna",
    "kor": 25,
    "varos": "Budapest"
}
print(my_dict["nev"])  # Anna

Szótár metódusai

  • keys(): A szótár kulcsait adja vissza.
  • values(): A szótár értékeit adja vissza.
  • items(): A kulcs-érték párokat adja vissza.
  • pop(): Egy elem törlése a kulcs alapján.
  • update(): Új kulcs-érték párokat ad a szótárhoz.
# Kulcsok lekérése
print(my_dict.keys())

# Értékek lekérése
print(my_dict.values())

# Kulcs-érték párok iterálása
for kulcs, ertek in my_dict.items():
    print(f"{kulcs}: {ertek}")

Szótár bővítése és törlése

Új kulcs-érték pár hozzáadása vagy meglévő elem törlése:

# Új kulcs-érték pár hozzáadása
my_dict["orszag"] = "Magyarország"

# Elem törlése
my_dict.pop("kor")
print(my_dict)
