4. Boolean és Listák

Boolean változó típus

A bool típusú változó két értéket vehet fel: True (igaz) vagy False (hamis).

Típuskényszerítés például:

valtozo = bool(1)  # True
valtozo2 = bool(0)  # False

Listák

A lista egy olyan változó, amelyben több értéket tárolhatunk egyszerre. A lista szögletes zárójelek között van definiálva.

gyumolcsok = ["alma", "körte", "barack"]

Indexelés

A listában lévő elemek indexelése mindig 0-tól kezdődik:

print(gyumolcsok[0])  # alma

Érték módosítása

A lista elemei módosíthatók az indexük alapján:

gyumolcsok[1] = "szilva"
print(gyumolcsok)  # ["alma", "szilva", "barack"]

A len() függvény

A len() függvény visszaadja a lista elemeinek számát:

print(len(gyumolcsok))  # 3

Elem hozzáadása

Új elemeket adhatunk hozzá a lista.append() vagy lista.insert() metódusokkal:

# Hozzáadás a lista végéhez
lista.append("szőlő")

# Hozzáadás meghatározott indexhez
lista.insert(1, "kompót")

Elem törlése

Törlés index alapján vagy érték szerint:

# Törlés érték szerint
lista.remove("alma")

# Törlés index szerint
del lista[0]

Lista és ciklus

Bejárhatjuk a lista elemeit egy for ciklussal:

for gyumolcs in gyumolcsok:
    print(gyumolcs)

Lista ellenőrzése

Ellenőrizhetjük, hogy egy elem szerepel-e a listában:

if "alma" in gyumolcsok:
    print("Az alma benne van a listában.")

Minimális és maximális érték

A min() és max() függvényekkel lekérhetjük a legkisebb és legnagyobb értéket:

szamok = [3, 5, 1, 8]
print(min(szamok))  # 1
print(max(szamok))  # 8
