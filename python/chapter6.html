6. Sztring Metódusok

Sztring-metódusok Áttekintése

A sztring metódusok segítségével sztring változókon végezhetünk különféle műveleteket, például kisbetűk nagybetűvé alakítása, szöveg középre igazítása stb.

Példák Sztring-metódusokra

  • capitalize(): Az első betűt nagybetűvé alakítja.
  • center(): Középre igazítja a szöveget egy megadott szélesség mentén.
  • startswith(): Megállapítja, hogy a szöveg a megadott karakterrel kezdődik-e.
  • endswith(): Ellenőrzi, hogy a szöveg a megadott karakterrel végződik-e.
  • find(): A karakterlánc első előfordulási helyét adja vissza.
  • rfind(): Az utolsó előfordulási helyet adja vissza.
  • isalnum(): Igaz, ha a szöveg minden karaktere alfanumerikus.
  • isalpha(): Igaz, ha a szövegben csak betűk vannak.
  • islower(): Igaz, ha minden betű kisbetű.
  • title(): Minden szó kezdőbetűjét nagybetűvé alakítja.
  • lower(): Minden betűt kisbetűvé alakít.
  • upper(): Minden betűt nagybetűvé alakít.
  • swapcase(): A kisbetűket nagybetűvé, a nagybetűket kisbetűvé alakítja.
  • replace(): Lecserél egy szövegrészt egy másikra.
  • join(): Egy iterálható objektum elemeit szöveggé fűzi.
  • lstrip(): Eltávolítja a szöveg elején található felesleges karaktereket.
  • rstrip(): Eltávolítja a szöveg végén található felesleges karaktereket.
  • strip(): Eltávolítja a szöveg elején és végén található felesleges karaktereket.
  • split(): Egy szöveget listaelemekre bont.

Példa Kódok

capitalize()

szoveg = "hello python"
print(szoveg.capitalize())  # "Hello python"

center()

szoveg = "banán"
print(szoveg.center(15))  # "     banán     "
print(szoveg.center(30, "-"))  # "-----------banán-----------"

find() és rfind()

szoveg = "alma körte barack"
print(szoveg.find("körte"))  # 5
print(szoveg.rfind("k"))  # 17

replace()

szoveg = "alma körte barack"
uj_szoveg = szoveg.replace("alma", "citrom")
print(uj_szoveg)  # "citrom körte barack"

split()

szoveg = "alma, körte, barack"
lista = szoveg.split(", ")
print(lista)  # ["alma", "körte", "barack"]

strip()

szoveg = "   alma   "
print(szoveg.strip())  # "alma"
