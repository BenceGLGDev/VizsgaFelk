Összefoglaló

Az alábbiakban összefoglaljuk a Python Kurzus során bemutatott főbb kódokat:

Adatbevitel

Ciklusok

Boolean és Listák

gyumolcsok = ["alma", "körte", "szilva"]

Random Modul

Sztring Metódusok

Tuple és Szótárak

Függvények és Eljárások

    print(f"Üdv, {nev}!")

Fájlkezelés

    f.write("Hello, világ!")

Objektumorientált Programozás

class Ember:
    def __init__(self, nev, kor):
        self.nev = nev
        self.kor = kor

    def koszont(self):
        print(f"Szia, {self.nev}!")

ember1 = Ember("Anna", 25)
ember1.koszont()
