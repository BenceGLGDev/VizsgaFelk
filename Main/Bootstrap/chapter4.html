4. Bootstrap Komponensek I

Bootstrap komponensek áttekintése

A Bootstrap egyik legnagyobb előnye a kész komponensek gazdag gyűjteménye, amelyek azonnal használhatók és reszponzívak.

Ebben a fejezetben a leggyakrabban használt komponensek első csoportját ismerjük meg:

  • Gombok és gomb csoportok
  • Navigációs sávok
  • Kártyák

Gombok (Buttons)

A Bootstrap előre definiált gombosztályai segítségével könnyen létrehozhatunk különböző stílusú és méretű gombokat.

Alapvető gombok

<button type="button" class="btn btn-primary">Elsődleges</button>
<button type="button" class="btn btn-secondary">Másodlagos</button>
<button type="button" class="btn btn-success">Sikeres</button>
<button type="button" class="btn btn-danger">Veszély</button>
<button type="button" class="btn btn-warning">Figyelmeztetés</button>
<button type="button" class="btn btn-info">Információ</button>
<button type="button" class="btn btn-light">Világos</button>
<button type="button" class="btn btn-dark">Sötét</button>
<button type="button" class="btn btn-link">Link</button>

Gomb méretek

A gomb méretét a btn-lg vagy btn-sm osztályokkal módosíthatjuk:

<button type="button" class="btn btn-primary btn-lg">Nagy gomb</button>
<button type="button" class="btn btn-primary">Normál gomb</button>
<button type="button" class="btn btn-primary btn-sm">Kis gomb</button>

Körvonalazott gombok

Az btn-outline-* osztállyal körvonalazott gombokat készíthetünk:

<button type="button" class="btn btn-outline-primary">Elsődleges</button>
<button type="button" class="btn btn-outline-secondary">Másodlagos</button>

Gomb csoportok

A btn-group osztállyal több gombot egy csoportba rendezhetünk:

<div class="btn-group" role="group" aria-label="Alapvető példa">
  <button type="button" class="btn btn-primary">Bal</button>
  <button type="button" class="btn btn-primary">Közép</button>
  <button type="button" class="btn btn-primary">Jobb</button>
</div>

Navigációs sávok (Navbars)

A navigációs sávok a weblapok egyik legfontosabb elemei, amelyek segítségével a felhasználók könnyedén navigálhatnak az oldalon.

Alapvető navigációs sáv

<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Weboldal neve</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" 
            data-bs-target="#navbarNav" aria-controls="navbarNav" 
            aria-expanded="false" aria-label="Toggle navigation">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" aria-current="page" href="#">Főoldal</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Szolgáltatások</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Rólunk</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Kapcsolat</a>
        </li>
      </ul>
    </div>
  </div>
</nav>

Navigációs sáv színváltozatai

A Bootstrap alapértelmezetten két színváltozatot kínál: navbar-light világos hátterű és navbar-dark sötét hátterű sávokhoz:

<!-- Világos navigációs sáv -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">...</nav>

<!-- Sötét navigációs sáv -->
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">...</nav>

<!-- Egyedi háttérszínű navigációs sáv -->
<nav class="navbar navbar-expand-lg navbar-dark bg-primary">...</nav>

Reszponzív viselkedés

A navigációs sáv reszponzív töréspontját a navbar-expand-* osztály határozza meg (sm, md, lg, xl, xxl):

<!-- lg képernyőméretnél nyílik ki -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">...</nav>

<!-- md képernyőméretnél nyílik ki -->
<nav class="navbar navbar-expand-md navbar-light bg-light">...</nav>

Kártyák (Cards)

A kártyák rugalmas tartalomelőállító komponensek, melyek segítségével rendszerezett formában jeleníthetjük meg az információkat.

Alapvető kártya

<div class="card" style="width: 18rem;">
  <img src="kep.jpg" class="card-img-top" alt="Kép leírása">
  <div class="card-body">
    <h5 class="card-title">Kártya címe</h5>
    <p class="card-text">Rövid leírás a kártyán belül.</p>
    <a href="#" class="btn btn-primary">Gomb</a>
  </div>
</div>

Kártya elrendezési lehetőségek

A kártyák különböző részeinek elhelyezése:

<div class="card">
  <div class="card-header">
    Kártya fejléc
  </div>
  <div class="card-body">
    <h5 class="card-title">Különleges cím</h5>
    <p class="card-text">Elsődleges kártya szövege.</p>
    <a href="#" class="btn btn-primary">Gomb</a>
  </div>
  <div class="card-footer text-muted">
    Kártya lábléc
  </div>
</div>

Kártya csoportok

Gyakorló Feladatok:

  1. Készíts egy oldalt, amelyen megjeleníted az összes Bootstrap gomb típust!
  2. Hozz létre egy reszponzív navigációs sávot, amely tartalmaz egy logót, négy menüpontot és egy keresőmezőt!
  3. Készíts egy termékoldalt három termék kártyával, amelyek tartalmazzák a termék nevét, leírását, árát és egy "Kosárba" gombot!
  4. Hozz létre egy gombcsoportot, amely tartalmaz három méretopciót (S, M, L) egy webshop számára!
