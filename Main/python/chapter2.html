2. Adatbevitel

Feladatok

  • koszones.py: Készíts egy nev változót, majd a változót felhasználva írd ki a képernyőre: "Üdvözöllek a programozás világában kedves Katalin!"
  • szamolas.py: Készíts két egész típusú változót (szam1, szam2), majd írd ki a képernyőre az összegüket, különbségüket, szorzatukat, hányadosukat egyetlen print segítségével!

Adat bekérése felhasználótól

A Python input() függvénye segítségével kérhetünk be adatot a felhasználótól. Példa:

nev = input("Mi a neved? ")
print("Üdvözöllek, " + nev + "!")

Az input működése

Fontos! Az input() függvényből érkező adat mindig string (karakterlánc) típusú. Ha számítási műveleteket szeretnénk végezni, akkor típus kényszerítés szükséges:

szam = int(input("Adj meg egy számot: "))
print(szam * 2)

Elágazások

A Python if, elif és else kulcsszavakkal támogatja a feltételes logikát:

szam = int(input("Adj meg egy számot: "))
if szam > 0:
    print("A szám pozitív.")
elif szam < 0:
    print("A szám negatív.")
else:
    print("A szám nulla.")
