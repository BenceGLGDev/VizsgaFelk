4. Gombok és Űrlapok

Gombok használata

A gombok interakciókat biztosítanak a felhasználó számára, például egy űrlap beküldésére vagy egy esemény indítására. Egy alap HTML gomb így néz ki:

<button>Kattints ide!</button>

Gombok stílusozása

CSS segítségével formázhatjuk a gombokat, például:

button {
    background-color: blue;
    color: white;
    padding: 10px 20px;
    border: none;
    cursor: pointer;
}
button:hover {
    background-color: darkblue;
}

Ez a kód egy kék gombot hoz létre, amely sötétebbé válik, ha az egér fölé kerül.

Űrlapok létrehozása

Az űrlapok segítségével a felhasználók adatokat küldhetnek be. Egy egyszerű HTML űrlap:

<form>
    <label for="name">Név:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <button type="submit">Küldés</button>
</form>

Ez egy egyszerű űrlap név és email mezővel, valamint egy beküldő gombbal.

Űrlapelemek formázása

A CSS segítségével az űrlapelemek megjelenése javítható:

input {
    width: 100%;
    padding: 8px;
    margin: 5px 0;
    border: 1px solid #ccc;
    border-radius: 4px;
}

Gyakorló Feladatok:

  1. Hozz létre egy gombot, amelyre kattintva egy JavaScript alert üzenet jelenik meg.
  2. Készíts egy űrlapot, amely tartalmaz egy név, email és jelszó mezőt.
  3. Stílusozd az űrlap elemeit, hogy jobban nézzenek ki.
