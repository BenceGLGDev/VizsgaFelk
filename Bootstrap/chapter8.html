8. Segédprogramok és Segédosztályok

Mi is az a Utility First megközelítés?

A Bootstrap egyik legnagyobb erőssége a segédosztályok (utilities) gazdag gyűjteménye, amelyek lehetővé teszik a gyakori stylusok gyors alkalmazását HTML elemekre külön CSS fájlok írása nélkül.

A Bootstrap 5.3 a "utility-first" megközelítést alkalmazza, amely lehetővé teszi a fejlesztők számára, hogy közvetlenül a HTML-ben formázzák az elemeket, anélkül hogy egyedi CSS osztályokat kellene létrehozniuk.

Spacing segédosztályok (margin, padding)

A Bootstrap egyszerű és következetes osztályrendszert kínál a térközök beállítására:

  • m-* - margin
  • p-* - padding

Irányok:

  • t - top (felső)
  • b - bottom (alsó)
  • s - start (bal oldal LTR nyelvekben)
  • e - end (jobb oldal LTR nyelvekben)
  • x - bal és jobb
  • y - felső és alsó
  • nincs prefix - mind a négy oldal

Méret értékek (0-tól 5-ig, valamint auto):

  • 0 - 0
  • 1 - 0.25rem (4px)
  • 2 - 0.5rem (8px)
  • 3 - 1rem (16px)
  • 4 - 1.5rem (24px)
  • 5 - 3rem (48px)
  • auto - auto

Spacing példák:

<div class="mb-4">Alsó margin: 1.5rem (24px)</div>
<div class="p-3">Minden oldalon padding: 1rem (16px)</div>
<div class="pt-5 pb-3">Felső padding: 3rem, alsó padding: 1rem</div>
<div class="mx-auto" style="width: 200px">Vízszintesen középre igazítva</div>
<div class="m-0">Nincs margin sehol</div>

Display tulajdonságok

A Bootstrap számos segédosztályt kínál a display CSS tulajdonság beállítására:

  • d-none - display: none
  • d-inline - display: inline
  • d-inline-block - display: inline-block
  • d-block - display: block
  • d-table - display: table
  • d-flex - display: flex
  • d-inline-flex - display: inline-flex
  • d-grid - display: grid

Reszponzív display osztályok:

A display tulajdonságok kombinálhatók a Bootstrap breakpointjaival a reszponzív viselkedés érdekében:

<div class="d-none d-md-block">
    Ez az elem csak közepes (md) és nagyobb képernyőkön jelenik meg
</div>

<div class="d-block d-lg-none">
    Ez az elem csak nagy (lg) méretnél kisebb képernyőkön jelenik meg
</div>

Közös használati esetek:

  • Elemek elrejtése vagy megjelenítése különböző képernyőméreteken
  • Alternatív navigáció mobil eszközökön
  • Különböző elrendezések használata eltérő képernyőméreteken

Flex segédosztályok

A Bootstrap átfogó flex segédosztályokat kínál a flexbox CSS tulajdonságainak kezeléséhez:

Flex konténer beállítások:

  • d-flex - display: flex
  • d-inline-flex - display: inline-flex
  • flex-row - flex-direction: row
  • flex-row-reverse - flex-direction: row-reverse
  • flex-column - flex-direction: column
  • flex-column-reverse - flex-direction: column-reverse

Justify Content (vízszintes igazítás):

  • justify-content-start - justify-content: flex-start
  • justify-content-end - justify-content: flex-end
  • justify-content-center - justify-content: center
  • justify-content-between - justify-content: space-between
  • justify-content-around - justify-content: space-around
  • justify-content-evenly - justify-content: space-evenly

Align Items (függőleges igazítás):

  • align-items-start - align-items: flex-start
  • align-items-end - align-items: flex-end
  • align-items-center - align-items: center
  • align-items-baseline - align-items: baseline
  • align-items-stretch - align-items: stretch

Flex példa:

<div class="d-flex justify-content-between align-items-center">
    <div>1. elem</div>
    <div>2. elem</div>
    <div>3. elem</div>
</div>

Ez a példa egy flexbox konténert hoz létre, ahol az elemek vízszintesen egyenlő távolságra vannak egymástól (space-between) és függőlegesen középre vannak igazítva.

Szín segédosztályok

A Bootstrap előre definiált színrendszert kínál, amelyet különböző elemekre alkalmazhatunk:

Szöveg színek:

  • text-primary - elsődleges szín
  • text-secondary - másodlagos szín
  • text-success - siker szín (általában zöld)
  • text-danger - veszély szín (általában piros)
  • text-warning - figyelmeztetés szín (általában sárga)
  • text-info - információ szín (általában világoskék)
  • text-light - világos szín
  • text-dark - sötét szín
  • text-body - alap szövegszín
  • text-muted - halványabb szövegszín
  • text-white - fehér szöveg
  • text-black-50 - félig átlátszó fekete szöveg
  • text-white-50 - félig átlátszó fehér szöveg

Háttérszínek:

  • bg-primary - elsődleges háttérszín
  • bg-secondary - másodlagos háttérszín
  • bg-success - siker háttérszín
  • bg-danger - veszély háttérszín
  • bg-warning - figyelmeztetés háttérszín
  • bg-info - információ háttérszín
  • bg-light - világos háttérszín
  • bg-dark - sötét háttérszín
  • bg-white - fehér háttér
  • bg-transparent - átlátszó háttér

Színkombinációk példa:

<div class="bg-primary text-white p-3">Elsődleges háttér, fehér szöveg</div>
<div class="bg-success text-white p-3">Siker háttér, fehér szöveg</div>
<div class="bg-light text-dark p-3">Világos háttér, sötét szöveg</div>
<span class="text-danger">Ez egy fontos figyelmeztetés</span>

Méretezési segédosztályok

A Bootstrap segédosztályokat kínál a szélesség és magasság beállításához:

Szélesség:

  • w-25 - 25% szélesség
  • w-50 - 50% szélesség
  • w-75 - 75% szélesség
  • w-100 - 100% szélesség
  • w-auto - auto szélesség

Magasság:

  • h-25 - 25% magasság
  • h-50 - 50% magasság
  • h-75 - 75% magasság
  • h-100 - 100% magasság
  • h-auto - auto magasság

Méretezési példa:

<div class="w-50">50% széles elem</div>
<div class="w-75 h-50">75% széles és 50% magas elem</div>
<img src="kep.jpg" class="w-100" alt="Reszponzív kép">

Keret (border) segédosztályok

A Bootstrap segédosztályokat kínál a keretek gyors kezeléséhez:

Alap keretek:

  • border - Keret minden oldalon
  • border-top - Keret csak felül
  • border-end - Keret csak jobbra
  • border-bottom - Keret csak alul
  • border-start - Keret csak balra

Keret színek:

  • border-primary, border-secondary, stb.

Keret vastagság:

  • border-0 - Nincs keret
  • border-1 - 1px vastag keret
  • border-2 - 2px vastag keret
  • border-3 - 3px vastag keret
  • border-4 - 4px vastag keret
  • border-5 - 5px vastag keret

Lekerekített sarkok:

  • rounded - Alapértelmezett lekerekítés
  • rounded-0 - Nincs lekerekítés
  • rounded-1 - Kicsi lekerekítés
  • rounded-2 - Közepes lekerekítés
  • rounded-3 - Nagy lekerekítés
  • rounded-circle - Kör alakú
  • rounded-pill - Pirula alakú
  • rounded-top, rounded-end, rounded-bottom, rounded-start - Csak bizonyos oldalon lekerekített

Keret példák:

<div class="border border-primary p-3 mb-3">Elsődleges színű kerettel</div>
<div class="border-top border-danger border-3 p-3 mb-3">Vastag vörös felső kerettel</div>
<div class="border rounded-pill border-success p-2 mb-3">Pirula alakú zöld kerettel</div>
<img src="profil.jpg" class="rounded-circle border border-dark" alt="Profilkép">

Árnyék segédosztályok

A Bootstrap árnyék segédosztályokat kínál a boxshadow alkalmazásához:

  • shadow-none - Nincs árnyék
  • shadow-sm - Kis árnyék
  • shadow - Normál árnyék
  • shadow-lg - Nagy árnyék

Árnyék példa:

<div class="shadow-sm p-3 mb-3 bg-white">Kis árnyékkal</div>
<div class="shadow p-3 mb-3 bg-white">Normál árnyékkal</div>
<div class="shadow-lg p-3 mb-3 bg-white">Nagy árnyékkal</div>

Pozícionálási segédosztályok

A Bootstrap segédosztályokat kínál az elemek pozícionálásához:

  • position-static - Statikus pozícionálás
  • position-relative - Relatív pozícionálás
  • position-absolute - Abszolút pozícionálás
  • position-fixed - Fix pozícionálás
  • position-sticky - Ragadós pozícionálás

Pozíció példa:

<div class="position-relative" style="height: 200px;">
    <div class="position-absolute top-0 start-0">Bal felső sarok</div>
    <div class="position-absolute top-0 end-0">Jobb felső sarok</div>
    <div class="position-absolute bottom-0 start-0">Bal alsó sarok</div>
    <div class="position-absolute bottom-0 end-0">Jobb alsó sarok</div>
    <div class="position-absolute top-50 start-50 translate-middle">Középre igazítva</div>
</div>

Gyakorló Feladatok:

  1. Hozz létre egy kartyát, ami 50% széles, középre igazított, árnyékkal és lekerekített sarkokkal rendelkezik!
  2. Készíts egy flexbox konténert, amiben három elem van: balra, középre és jobbra igazítva!
  3. Hozz létre egy képet, ami reszponzív, kerek alakú, és kerettel rendelkezik!
  4. Készíts egy elemet, ami csak mobilon jelenik meg (sm breakpoint alatt), és egy másikat, ami csak asztali eszközökön látható (lg breakpoint felett)!
  5. Hozz létre egy információs blokkot kék háttérrel, fehér szöveggel, megfelelő padding-gel és lekerekített sarkokkal!

Összefoglalás

A Bootstrap segédosztályok rendkívül hasznosak a gyors fejlesztéshez és a konzisztens megjelenés biztosításához. Ezek használatával:

  • Gyorsabban fejleszthetsz, mivel kevesebb egyedi CSS-t kell írnod
  • Konzisztensebb lesz a felületed, mivel előre definiált értékeket használsz
  • Reszponzívabb és rugalmasabb lesz az oldalad
  • Könnyebben módosíthatod a megjelenést később

A segédosztályok kombinálásával rendkívül sokféle elrendezést és dizájnt hozhatsz létre anélkül, hogy külön CSS fájlokat kellene írnod.
