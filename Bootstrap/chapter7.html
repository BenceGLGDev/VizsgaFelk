7. Képek és Médiaelemek

Képek a Bootstrap 5.3-ban

A képek és más médiaelemek elengedhetetlen részei a modern weboldalaknak. A Bootstrap 5.3 számos segédosztályt kínál, amelyek segítenek a képek és más médiaelemek megfelelő elrendezésében és formázásában.

Reszponzív képek

A Bootstrap legfontosabb képkezelési tulajdonsága a reszponzivitás támogatása. Az img-fluid osztály hozzáadásával bármely kép automatikusan reszponzívvá válik:

<img src="kep.jpg" class="img-fluid" alt="Reszponzív kép">

Ez a kód a következő CSS tulajdonságokat alkalmazza a képre:

img-fluid {
  max-width: 100%;
  height: auto;
}

Ennek eredményeként a kép sosem lesz szélesebb a szülő elemnél, mérete automatikusan igazodik a képernyő méretéhez, miközben megőrzi az eredeti képarányt.

Kép miniatűrök (Thumbnails)

A img-thumbnail osztállyal keretezett, lekerekített sarkú miniatűröket készíthetsz:

<img src="kep.jpg" class="img-thumbnail" alt="Miniatűr kép">

Ez egy vékony szürke keretet és lekerekített sarkokat ad a képhez, ami különösen hasznos galériák vagy terméklisták esetén.

Képek igazítása

A Bootstrap segédosztályaival könnyen igazíthatod a képeket:

Szöveg köré igazítás

<img src="kep.jpg" class="float-start" alt="Balra igazított kép">
<p>A szöveg a kép mellett fog megjelenni...</p>

<img src="kep.jpg" class="float-end" alt="Jobbra igazított kép">
<p>A szöveg a kép mellett fog megjelenni...</p>

Középre igazítás

<div class="text-center">
  <img src="kep.jpg" class="img-fluid" alt="Középre igazított kép">
</div>

Blokkszintű, automatikus margókkal

<img src="kep.jpg" class="d-block mx-auto" alt="Középre igazított kép">

Kép alakzatok

A Bootstrap 5.3-ban különböző alakzatokat adhatsz a képeknek a segédosztályok használatával:

Lekerekített sarkok

<img src="kep.jpg" class="rounded" alt="Lekerekített sarkú kép">

Kör alakú kép

<img src="kep.jpg" class="rounded-circle" alt="Kör alakú kép">

A kör alakú képeknél fontos, hogy az eredeti kép négyzet alakú legyen a legjobb eredmény érdekében.

Lekerekített sarkok különböző irányokban

<!-- Csak felül lekerekített -->
<img src="kep.jpg" class="rounded-top" alt="Felül lekerekített kép">

<!-- Csak jobb oldalon lekerekített -->
<img src="kep.jpg" class="rounded-end" alt="Jobb oldalon lekerekített kép">

<!-- Csak alul lekerekített -->
<img src="kep.jpg" class="rounded-bottom" alt="Alul lekerekített kép">

<!-- Csak bal oldalon lekerekített -->
<img src="kep.jpg" class="rounded-start" alt="Bal oldalon lekerekített kép">

Picture elem használata

A Bootstrap jól működik a HTML5 <picture> elemmel, ami lehetővé teszi különböző képek betöltését különböző eszközökön:

<picture>
  <source srcset="nagy-kep.jpg" media="(min-width: 992px)">
  <source srcset="kozepes-kep.jpg" media="(min-width: 576px)">
  <img src="kicsi-kep.jpg" class="img-fluid" alt="Reszponzív kép">
</picture>

Ez a kód különböző képeket tölt be a képernyő méretétől függően:

  • 992px feletti képernyőn a "nagy-kep.jpg" töltődik be
  • 576px és 992px között a "kozepes-kep.jpg" töltődik be
  • 576px alatt a "kicsi-kep.jpg" töltődik be

Figure elemek

A Bootstrap megfelelő stílusokat biztosít a HTML5 <figure> elemhez, amely képek és feliratok együttes megjelenítésére szolgál:

<figure class="figure">
  <img src="kep.jpg" class="figure-img img-fluid rounded" alt="Képfelirat példa">
  <figcaption class="figure-caption text-center">Ez egy középre igazított képfelirat.</figcaption>
</figure>

A figure-caption a felirat szövegét formázza, és alapértelmezés szerint balra igazítja. A text-center vagy text-end osztályokkal változtathatod az igazítást.

Feliratok szövegszínének módosítása

<!-- Alapértelmezett (szürke) felirat -->
<figcaption class="figure-caption">Alapértelmezett szürke szöveg</figcaption>

<!-- Fehér felirat sötét háttéren -->
<figcaption class="figure-caption text-white">Fehér szöveg</figcaption>

Képek kártyákban

A Bootstrap kártya (card) komponense kiválóan alkalmas képek és kapcsolódó tartalmak megjelenítésére:

<div class="card" style="width: 18rem;">
  <img src="kep.jpg" class="card-img-top" alt="Kártya kép">
  <div class="card-body">
    <h5 class="card-title">Kártya címe</h5>
    <p class="card-text">Ez egy példa szöveg a kártyához kapcsolódóan.</p>
    <a href="#" class="btn btn-primary">Gomb</a>
  </div>
</div>

A card-img-top osztály a képet a kártya tetejére helyezi. Használhatod a card-img-bottom osztályt is, ha a képet a kártya aljára szeretnéd helyezni.

Kép kártya háttérként

<div class="card text-white">
  <img src="hatter-kep.jpg" class="card-img" alt="Háttérkép">
  <div class="card-img-overlay">
    <h5 class="card-title">Kártya címe</h5>
    <p class="card-text">Ez a szöveg a kép felett jelenik meg.</p>
  </div>
</div>

Itt a card-img-overlay osztály lehetővé teszi, hogy a kártya tartalma a kép fölött jelenjen meg.

Videók beágyazása

A Bootstrap reszponzív segédosztályokat kínál videók beágyazásához is:

Reszponzív videó beágyazása

<div class="ratio ratio-16x9">
  <iframe src="https://www.youtube.com/embed/zpOULjyy-n8" title="YouTube videó" allowfullscreen></iframe>
</div>

A ratio osztály és annak változatai biztosítják, hogy a beágyazott tartalom megtartsa a megfelelő képarányt, miközben reszponzív módon méretezik.

Különböző képarányok

A Bootstrap különböző képarányokat támogat:

  • ratio-1x1: Négyzet alakú (1:1)
  • ratio-4x3: Klasszikus TV (4:3)
  • ratio-16x9: Szélesvásznú (16:9)
  • ratio-21x9: Ultraszéles (21:9)
<!-- 4:3 képarányú videó -->
<div class="ratio ratio-4x3">
  <iframe src="https://www.youtube.com/embed/zpOULjyy-n8" title="YouTube videó" allowfullscreen></iframe>
</div>

Egyéni képarányok

Ha egyéni képarányt szeretnél használni, azt egyszerűen megteheted saját CSS stílusokkal:

<div class="ratio" style="--bs-aspect-ratio: 50%;">
  <iframe src="https://www.example.com/embed/video" title="Beágyazott videó"></iframe>
</div>

A --bs-aspect-ratio CSS változó segítségével bármilyen egyéni képarányt beállíthatsz.

Képek modális ablakok használatával

A Bootstrap modális ablakait használhatod egyszerű lightbox hatás eléréséhez, külső könyvtárak nélkül:

<!-- Miniatűr kép, amely megnyitja a modális ablakot -->
<img src="kep-kicsi.jpg" class="img-thumbnail" alt="Miniatűr kép" data-bs-toggle="modal" data-bs-target="#kepModal">

<!-- Modális ablak a nagyított képpel -->
<div class="modal fade" id="kepModal" tabindex="-1" aria-labelledby="kepModalLabel" aria-hidden="true">
  <div class="modal-dialog modal-dialog-centered modal-lg">
    <div class="modal-content">
      <div class="modal-body p-0">
        <img src="kep-nagy.jpg" class="img-fluid" alt="Nagyított kép">
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Bezárás</button>
      </div>
    </div>
  </div>
</div>

Ebben a példában a kép miniatűrre kattintva megnyílik egy modális ablak, amely a kép nagyobb verzióját mutatja.

Képek SEO optimalizálása

A képek megfelelő használata nemcsak a felhasználói élmény, hanem a keresőoptimalizálás szempontjából is fontos:

Fontos gyakorlatok:

  • Mindig használj leíró alt attribútumokat a képeken.
  • Használj beszédes fájlneveket (pl. "piros-polo.jpg" jobb, mint "DSC12345.jpg").
  • Optimalizáld a képek fájlméretét a gyorsabb betöltés érdekében.
  • Használd a <picture> elemet vagy srcset attribútumot különböző felbontásokhoz.
  • Lazy loading használata a nagy képgalériáknál: <img loading="lazy" ...>

Képszűrők CSS-sel

A Bootstrap nem tartalmaz beépített képszűrőket, de egyszerűen létrehozhatsz saját CSS osztályokat erre a célra:

<style>
/* Szürkeárnyalatos szűrő */
.filter-grayscale {
  filter: grayscale(100%);
}

/* Szépia szűrő */
.filter-sepia {
  filter: sepia(100%);
}

/* Fényerő módosítás */
.filter-brightness {
  filter: brightness(150%);
}

/* Kontraszt módosítás */
.filter-contrast {
  filter: contrast(200%);
}

/* Átmenet szűrők között hover eseményre */
.filter-transition {
  filter: grayscale(100%);
  transition: filter 0.3s;
}
.filter-transition:hover {
  filter: grayscale(0%);
}
</style>

<!-- Szűrők használata -->
<img src="kep.jpg" class="img-fluid filter-grayscale" alt="Szürkeárnyalatos kép">
<img src="kep.jpg" class="img-fluid filter-transition" alt="Átmenetes szűrővel ellátott kép">
