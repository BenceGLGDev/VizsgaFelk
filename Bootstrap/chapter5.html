5. Navigációs elemek

Miért fontos a navigáció?

A navigáció elengedhetetlen egy weboldalon, mert segíti a felhasználókat az információ gyors elérésében. Egy jól megtervezett menü intuitív és könnyen használható.

Egyszerű navigációs sáv

Egy alapvető navigációs sáv így néz ki HTML-ben:

<nav>
    <ul>
        <li><a href="#">Kezdőlap</a></li>
        <li><a href="#">Rólunk</a></li>
        <li><a href="#">Kapcsolat</a></li>
    </ul>
</nav>

Stílusozás CSS segítségével

A menü megjelenését CSS-sel lehet formázni:

nav ul {
    list-style: none;
    padding: 0;
    display: flex;
    background-color: #333;
}
nav ul li {
    margin: 0;
    padding: 10px 20px;
}
nav ul li a {
    color: white;
    text-decoration: none;
}

Ez egy vízszintes, sötét hátterű menüt eredményez, amely jól látható és könnyen használható.

Reszponzív navigáció

Ahhoz, hogy a navigáció mobilbarát legyen, használhatunk CSS média lekérdezéseket:

@media (max-width: 768px) {
    nav ul {
        flex-direction: column;
    }
}

Ez a kód a navigációs elemeket egymás alá rendezi kis képernyőkön.

Gyakorló Feladatok:

  1. Hozz létre egy új navigációs menüt legalább 4 menüponttal.
  2. Stílusozd a menüt, hogy vizuálisan kiemelkedjen.
  3. Adj hozzá egy hamburger menüt kis képernyőkre.
