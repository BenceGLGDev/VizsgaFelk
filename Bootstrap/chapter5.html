5. Bootstrap Komponensek II

További Bootstrap komponensek

Az előző fejezetben megismertük a gombok, navigációs sávok és kártyák használatát. Most folytatjuk a Bootstrap komponensek felfedezését a következőkkel:

  • Űrlapok és űrlapelemek

Űrlapok és űrlapelemek (Forms)

Validáció

Bootstrap beépített validációs visszajelzést is kínál:

<form class="needs-validation" novalidate>
  <div class="mb-3">
    <label for="emailValidacio" class="form-label">Email cím</label>
    <input type="email" class="form-control" id="emailValidacio" required>
    <div class="valid-feedback">
      Megfelelő!
    </div>
    <div class="invalid-feedback">
      Kérjük, adj meg egy érvényes email címet!
    </div>
  </div>
  <div class="mb-3">
    <label for="jelszoValidacio" class="form-label">Jelszó</label>
    <input type="password" class="form-control" id="jelszoValidacio" required>
    <div class="invalid-feedback">
      A jelszó megadása kötelező!
    </div>
  </div>
  <button class="btn btn-primary" type="submit">Küldés</button>
</form>

<script>
// JavaScript a validáció aktiválásához
(function () {
  'use strict'
  // Összes form kiválasztása
  var forms = document.querySelectorAll('.needs-validation')
  // Loop, ami megakadályozza a küldést és validálja a mezőket
  Array.prototype.slice.call(forms)
    .forEach(function (form) {
      form.addEventListener('submit', function (event) {
        if (!form.checkValidity()) {
          event.preventDefault()
          event.stopPropagation()
        }
        form.classList.add('was-validated')
      }, false)
    })
})()
</script>

Modális ablakok és tooltipek

A modális ablakok és tooltipek interaktív elemek, amelyek további információt vagy funkciót kínálnak a felhasználónak.

Modális ablak (Modal)

<!-- Gomb a modális ablak megnyitásához -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#peldaModal">
  Modális ablak megnyitása
</button>

<!-- Modális ablak -->
<div class="modal fade" id="peldaModal" tabindex="-1" aria-labelledby="peldaModalLabel" aria-hidden="true">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title" id="peldaModalLabel">Modális ablak címe</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal" aria-label="Bezárás"></button>
      </div>
      <div class="modal-body">
        Ez a modális ablak tartalma.
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Bezárás</button>
        <button type="button" class="btn btn-primary">Mentés</button>
      </div>
    </div>
  </div>
</div>

Tooltip (eszköztipp)

A tooltipek segítségével további információt jeleníthetünk meg, amikor a felhasználó egy elem fölé viszi az egeret:

<button type="button" class="btn btn-secondary" 
        data-bs-toggle="tooltip" data-bs-placement="top" 
        title="Ez egy tooltip szövege">
  Tooltip felül
</button>

<button type="button" class="btn btn-secondary" 
        data-bs-toggle="tooltip" data-bs-placement="right" 
        title="Ez egy tooltip szövege">
  Tooltip jobbra
</button>

<script>
// Tooltipek inicializálása
var tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'))
var tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
  return new bootstrap.Tooltip(tooltipTriggerEl)
})
</script>

Popover

A popoverek a tooltipekhez hasonlóak, de több információt tartalmazhatnak:

<button type="button" class="btn btn-lg btn-danger" 
        data-bs-toggle="popover" 
        title="Popover cím" 
        data-bs-content="Itt van a popover tartalma. Több szöveget is megjeleníthet.">
  Kattints a popover megjelenítéséhez
</button>

<script>
// Popoverek inicializálása
var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'))
var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
  return new bootstrap.Popover(popoverTriggerEl)
})
</script>

Collapse (összecsukható tartalom)

Az összecsukható tartalommal helyet takaríthatunk meg, és a felhasználókra bízhatjuk, mely tartalmat szeretnék megtekinteni:

<!-- Egyszerű collapse -->
<p>
  <button class="btn btn-primary" type="button" data-bs-toggle="collapse" data-bs-target="#collapseExample" aria-expanded="false" aria-controls="collapseExample">
    Kattints a tartalom megjelenítéséhez
  </button>
</p>
<div class="collapse" id="collapseExample">
  <div class="card card-body">
    Ez egy rejtett tartalom, ami csak akkor jelenik meg, amikor a felhasználó a gombra kattint.
  </div>
</div>

Accordion (harmonika)

Az accordion egy speciális collapse komponens, amely több összecsukható részt tartalmaz:

JavaScript inicializálás

Fontos megjegyezni, hogy számos Bootstrap komponens JavaScript-et igényel a működéshez. Két módon inicializálhatjuk ezeket:

1. HTML attribútumok használatával (data-* attribútumok)

<button data-bs-toggle="modal" data-bs-target="#peldaModal">
  Modális ablak megnyitása
</button>

2. JavaScript használatával

// Modal inicializálása
var myModal = new bootstrap.Modal(document.getElementById('peldaModal'))

// Modal megjelenítése
myModal.show()

// Modal elrejtése
myModal.hide()

Gyakorló Feladatok:

  1. Készíts egy regisztrációs űrlapot, amely tartalmaz szövegmezőket, jelszómezőt, lenyíló listát és rádiógombokat!
  2. Hozz létre egy modális ablakot, amely egy termék részletes információit jeleníti meg, amikor a felhasználó a "Részletek" gombra kattint!
  3. Készíts egy képgalériát carousel komponens segítségével, amely automatikusan vált a képek között!
  4. Hozz létre egy GYIK (Gyakran Ismételt Kérdések) oldalt accordion komponens segítségével!
  5. Adj tooltip magyarázatokat egy űrlap mezőihez, amelyek segítenek a felhasználónak megérteni, mit kell beírnia!
