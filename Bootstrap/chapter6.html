6. Táblázatok és Listák

Táblázatok a Bootstrap 5.3-ban

A táblázatok az adatok rendszerezett megjelenítésének alapvető eszközei. A Bootstrap több beépített osztályt kínál a táblázatok formázásához és a reszponzivitás biztosításához.

Alap táblázat

A Bootstrap alapvető táblázat stílusának használatához egyszerűen add hozzá a table osztályt a <table> elemhez:

<table class="table">
  <thead>
    <tr>
      <th>#</th>
      <th>Név</th>
      <th>Email</th>
      <th>Szerepkör</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1</td>
      <td>Nagy János</td>
      <td>janos@pelda.hu</td>
      <td>Admin</td>
    </tr>
    <tr>
      <td>2</td>
      <td>Kiss Éva</td>
      <td>eva@pelda.hu</td>
      <td>Felhasználó</td>
    </tr>
  </tbody>
</table>

Táblázat variációk

A Bootstrap különböző táblázat stílusokat kínál, amelyek a table osztály mellett használhatók:

Osztály Leírás
table-striped Váltakozó színű sorokat ad a táblázatnak, megkönnyítve az olvashatóságot.
table-bordered Keretet ad minden cellához a táblázatban.
table-hover Kiemelő effektet ad az egérrel rámutatott soroknak.
table-sm Kompaktabb táblázatot eredményez kisebb padding értékekkel.
table-dark Sötét háttérrel és világos szöveggel jeleníti meg a táblázatot.

Ezek az osztályok kombinálhatók is. Például:

<table class="table table-striped table-hover table-bordered">
  <!-- táblázat tartalma -->
</table>

Kontextuális osztályok táblázatokhoz

Kontextuális osztályokkal kiemelheted a táblázat egyes sorait vagy celláit:

Osztály Leírás
table-primary Kék kiemelés (elsődleges szín)
table-secondary Szürke kiemelés (másodlagos szín)
table-success Zöld kiemelés (siker jelzése)
table-danger Piros kiemelés (veszély/hiba jelzése)
table-warning Sárga kiemelés (figyelmeztetés)
table-info Világoskék kiemelés (információ)
table-light Világos háttér
table-dark Sötét háttér

Ezeket az osztályokat használhatod a <tr>, <th> vagy <td> elemeken:

<tr class="table-success">
  <td>Sikeres művelet</td>
  <td>...</td>
</tr>
<tr>
  <td class="table-danger">Hibás adat</td>
  <td>...</td>
</tr>

Reszponzív táblázatok

A Bootstrap lehetővé teszi a táblázatok reszponzívvá tételét is. Ehhez csomagold a táblázatot egy .table-responsive osztályú div elembe:

<div class="table-responsive">
  <table class="table">
    <!-- táblázat tartalma -->
  </table>
</div>

Töréspontok szerint is meghatározhatod a reszponzivitást:

  • table-responsive-sm: csak kisebb képernyőkön (576px alatt) görget
  • table-responsive-md: 768px alatt
  • table-responsive-lg: 992px alatt
  • table-responsive-xl: 1200px alatt
  • table-responsive-xxl: 1400px alatt

Táblázat fejléc opciók

A thead-light és thead-dark osztályokkal formázhatod a táblázat fejlécét:

<table class="table">
  <thead class="thead-dark">
    <tr>
      <th>#</th>
      <th>Név</th>
      <!-- további fejléc cellák -->
    </tr>
  </thead>
  <tbody>
    <!-- táblázat sorai -->
  </tbody>
</table>

Lista csoportok (List Groups)

A Bootstrap lista csoportjai rugalmas és hatékony komponensek, amelyek segítségével rendszerezett listákat, navigációt vagy akár komplexebb tartalmakat jeleníthetsz meg.

Alapvető lista csoport

<ul class="list-group">
  <li class="list-group-item">Első elem</li>
  <li class="list-group-item">Második elem</li>
  <li class="list-group-item">Harmadik elem</li>
</ul>

Aktív és letiltott elemek

<ul class="list-group">
  <li class="list-group-item active">Aktív elem</li>
  <li class="list-group-item">Normál elem</li>
  <li class="list-group-item disabled">Letiltott elem</li>
</ul>

Lista csoport variációk

Linkek és gombok lista csoportban

Használhatsz <a> vagy <button> elemeket is <li> helyett:

<div class="list-group">
  <a href="#" class="list-group-item list-group-item-action active">Aktív link</a>
  <a href="#" class="list-group-item list-group-item-action">Link elem</a>
  <button type="button" class="list-group-item list-group-item-action">Gomb elem</button>
</div>

Kontextuális osztályok lista csoporthoz

A táblázatokhoz hasonlóan a lista csoportok is használhatnak színkódolást:

<ul class="list-group">
  <li class="list-group-item list-group-item-primary">Elsődleges</li>
  <li class="list-group-item list-group-item-secondary">Másodlagos</li>
  <li class="list-group-item list-group-item-success">Siker</li>
  <li class="list-group-item list-group-item-danger">Veszély</li>
  <li class="list-group-item list-group-item-warning">Figyelmeztetés</li>
  <li class="list-group-item list-group-item-info">Információ</li>
  <li class="list-group-item list-group-item-light">Világos</li>
  <li class="list-group-item list-group-item-dark">Sötét</li>
</ul>

Keret nélküli lista

Az list-group-flush osztállyal keret nélküli lista csoportot készíthetsz:

<ul class="list-group list-group-flush">
  <li class="list-group-item">Első elem</li>
  <li class="list-group-item">Második elem</li>
  <li class="list-group-item">Harmadik elem</li>
</ul>

Vízszintes lista csoport

A list-group-horizontal osztállyal vízszintes elrendezésű lista csoportot készíthetsz:

<ul class="list-group list-group-horizontal">
  <li class="list-group-item">Első elem</li>
  <li class="list-group-item">Második elem</li>
  <li class="list-group-item">Harmadik elem</li>
</ul>

Töréspontokat is használhatsz a lista vízszintes elrendezéséhez:

  • list-group-horizontal-sm: 576px felett
  • list-group-horizontal-md: 768px felett
  • list-group-horizontal-lg: 992px felett
  • list-group-horizontal-xl: 1200px felett
  • list-group-horizontal-xxl: 1400px felett

Összetett tartalom lista csoportokban

A lista csoportokban összetettebb tartalmakat is megjeleníthetsz:

<div class="list-group">
  <a href="#" class="list-group-item list-group-item-action">
    <div class="d-flex w-100 justify-content-between">
      <h5 class="mb-1">Lista elem fejléce</h5>
      <small>3 napja</small>
    </div>
    <p class="mb-1">Itt található a lista elem fő tartalma, ami hosszabb leírást is tartalmazhat.</p>
    <small>További részletek.</small>
  </a>
  <!-- további lista elemek -->
</div>

Jelölőnégyzetek és rádiógombok lista csoportokban

A Bootstrap 5.3 támogatja a jelölőnégyzetek és rádiógombok közvetlen használatát lista csoportokban:

<ul class="list-group">
  <li class="list-group-item">
    <input class="form-check-input me-1" type="checkbox" value="" id="firstCheckbox">
    <label class="form-check-label" for="firstCheckbox">Első jelölőnégyzet</label>
  </li>
  <li class="list-group-item">
    <input class="form-check-input me-1" type="checkbox" value="" id="secondCheckbox">
    <label class="form-check-label" for="secondCheckbox">Második jelölőnégyzet</label>
  </li>
</ul>

Jelvények lista csoportokban

A badge komponenst használhatod számláló vagy státusz jelzésére lista csoportokban:

<ul class="list-group">
  <li class="list-group-item d-flex justify-content-between align-items-center">
    Új üzenetek
    <span class="badge bg-primary rounded-pill">14</span>
  </li>
  <li class="list-group-item d-flex justify-content-between align-items-center">
    Feladatok
    <span class="badge bg-warning rounded-pill">2</span>
  </li>
</ul>

Gyakorló Feladatok:

  1. Készíts egy reszponzív, csíkozott táblázatot, amely egy osztály tanulóinak nevét, életkorát és tantárgyait tartalmazza!
  2. Hozz létre egy lista csoportot, amelyben a lista elemek különböző színkódolással rendelkeznek a fontosságuk alapján!
  3. Készíts egy táblázatot, amelyben minden sor más színű, és a fejléc sötét háttérrel rendelkezik!
  4. Hozz létre egy vízszintes lista csoportot, amely 768px felett vízszintesen, alatta függőlegesen jelenik meg!
  5. Készíts egy lista csoportot jelölőnégyzetekkel, ahol az egyik elem már előre be van jelölve!
