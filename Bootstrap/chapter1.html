1. Bevezetés a Bootstrap-be

A Bootstrap egy népszerű CSS keretrendszer, amely segít reszponzív és modern weboldalakat létrehozni.

Bootstrap Használata

A legegyszerűbb módja a Bootstrap használatának, ha a CDN-en keresztül hivatkozzuk:


<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

Első Bootstrap Oldal

Készítsünk egy alapvető Bootstrap oldalt:


<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Példa</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container">
        <h1>Hello, Bootstrap!</h1>
    </div>
</body>
</html>

Gyakorló Feladatok

  1. Hozz létre egy új Bootstrap HTML fájlt és adj hozzá egy h1 címsort.
  2. Használj egy Bootstrap osztályt a szöveg formázására.
  3. Próbáld ki a container és row osztályokat.
