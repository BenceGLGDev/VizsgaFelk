1. Bevezetés a Bootstrap 5.3-ba

A példák nem mutatják ki jól a kódot, legjobb ha magad is kipróbálod őket!


Mi is az a Bootstrap?

Röviden: egy népszerű, ingyenes és nyílt forráskódú CSS keretrendszer, amelyet Twitter fejlesztői hoztak létre 2011-ben.

A Bootstrap célja, hogy megkönnyítse a reszponzív, mobilbarát weboldalak fejlesztését.

  • Előre elkészített komponensek gyűjteménye
  • Reszponzív grid rendszer
  • JavaScript bővítmények
  • Egységes megjelenés különböző böngészőkön

Miért érdemes a Bootstrap-et használni?

  • Időmegtakarítás: Nem kell minden komponenst nulláról megírni.
  • Reszponzivitás: Automatikusan alkalmazkodik különböző képernyőméretekhez.
  • Konzisztencia: Egységes megjelenés az összes modern böngészőben.
  • Testreszabhatóság: Saját igényekhez alakítható.
  • Aktív közösség: Rengeteg dokumentáció, tutorial és példa elérhető.

Hogyan használjuk a Bootstrap-et?

Két fő módszer van a Bootstrap beépítésére:

1. CDN (Content Delivery Network) használata

<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- JavaScript Bundle with Popper -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

2. Letöltés és helyi telepítés

A Bootstrap CSS és JS fájlok letölthetők a hivatalos oldalról: https://getbootstrap.com/docs/5.3/getting-started/download/

Első Bootstrap oldal

<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Első Bootstrap Oldal</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container">
        <h1>Helló, Bootstrap!</h1>
        <p>Ez az első Bootstrap oldalam.</p>
        <button class="btn btn-primary">Kattints rám!</button>
    </div>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Fontos:

A Bootstrap JavaScript komponensei a jQuery helyett natív JavaScript-et használnak Bootstrap 5-től. Nincs szükség külön jQuery betöltésére!

Bootstrap Grid rendszer alapkoncepció

A Bootstrap 12 oszlopos, flexbox alapú grid rendszert használ:

  • A .container vagy .container-fluid tartalmazza a teljes grid-et.
  • A .row sorokat hoz létre.
  • Az oszlopok .col-* osztályokkal hozhatók létre.
<div class="container">
    <div class="row">
        <div class="col-sm-6">Fél szélesség kis képernyőktől</div>
        <div class="col-sm-6">Fél szélesség kis képernyőktől</div>
    </div>
</div>
